# prueba-vs

Aplicación web simple para ingresar y mostrar texto en pantalla.

## Características

- Interfaz intuitiva en español
- Campo de texto para ingresar contenido
- Botón "Mostrar Texto" para visualizar el texto ingresado
- Botón "Limpiar" para borrar todo
- Diseño responsive y moderno con gradiente de fondo
- Atajos de teclado: Ctrl+Enter para mostrar el texto

## Cómo usar

1. Abre el archivo `index.html` en tu navegador web
2. Escribe el texto que desees en el campo de texto
3. Haz clic en "Mostrar Texto" o presiona Ctrl+Enter
4. El texto aparecerá en la sección "Texto en pantalla"
5. Usa el botón "Limpiar" para borrar todo y empezar de nuevo

## Archivos

- `index.html` - Estructura HTML de la aplicación
- `styles.css` - Estilos y diseño visual
- `script.js` - Funcionalidad JavaScript
